    Other Tools - Assignment 3

    Other Tools that we utilised in this assignment were as followed:

    • Android Studio - Android Studio was used for the mockup and an APK was created for the artefact.This was the choice as the project is built on using Android studio for the application mentioned in the overview.
    • Pixelformer. - Pixelformer played a part in the creation of images (bullet points) for the website as well as tailoring our photos for the site. Pixelformer is a free to use image creation software with the ability to create transparent Alpha channel images for design.
    • Atom - Was used to code the repository page as well as the CSS. This was accessed via GITHUB desktop and allowed for easy changes and Pushes to the repository.
    • GitHub Desktop - Used for the pushing of the website to our team repository. This was used inline with Atom to aid in the creation of the web site.
